# Game Design Index

This area is for game-agnostic mechanics research, capability requests, UI notes, content notes, and development-method notes.

Game design notes may be written during hardware bring-up.

They must not redefine Platform behavior.

## Allowed Content

- mechanics concepts
- UI flow concepts
- abstract capability needs
- content production notes
- balancing and pacing research
- constraints discovered from Platform or Engine docs

## Not Allowed

- pin choices
- direct HAL/LL usage
- DMA policy
- RTOS ownership rules
- sleep-mode policy
- storage ownership
- peripheral fault handling
