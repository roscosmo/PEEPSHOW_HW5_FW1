# Memory Reports

Record memory-budget evidence here.

Each report should include:

- firmware commit
- build type
- map file artifact
- per-thread stack budgets
- queue storage budgets
- retained RAM use
- flash region use
- package/install path flash use
- violations or follow-up actions
