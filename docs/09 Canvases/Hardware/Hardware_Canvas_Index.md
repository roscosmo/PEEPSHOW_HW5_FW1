# Hardware Canvas Index

This folder contains visual Obsidian canvases for hardware-oriented thinking.

Authoritative hardware behavior still lives in [[Hardware_Index]] and the linked Platform contracts. Canvases are navigation, annotation, review, and spatial reasoning tools.

## Canvases

- [[HW5_Hardware_Visual_Map]]

## Intended Uses

- schematic overview
- PCB layout plot
- housing dimensions and mechanical constraints
- board-to-enclosure relationships
- bring-up annotation
- signal-flow review
- hardware/software boundary review

## Asset Location

Put exported visual files under:

- `docs/assets/hardware/hw5/schematic/`
- `docs/assets/hardware/hw5/pcb/`
- `docs/assets/hardware/hw5/housing/`
- `docs/assets/hardware/hw5/photos/`
- `docs/assets/hardware/hw5/reference-renders/`

See [[Hardware_Asset_Export_Guide]].

## Rule

Canvas annotations may point to hardware contracts.

Canvas annotations must not become the only source of truth for pin ownership, power policy, wake behavior, or bring-up evidence.
