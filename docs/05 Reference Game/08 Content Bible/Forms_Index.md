## Form Entry Requirements

Each form note should define:

- unlock condition
- visible identity markers
- retained slime traits
- traversal affordance
- social affordance
- combat affordance
- pet simulation behavior
- transformation pressure
- failure or instability behavior
- art/audio needs
- required Engine capabilities
