# Rituals Index

Rituals are recurring, memorable, time-bound events.

They should feel rare, deliberate, and emotionally charged rather than like chores.

## Ritual Entry Requirements

Each ritual note should define:

- real-world time window
- world preconditions
- discovery path
- required form or relationship state
- player input sequence
- failure behavior
- repeatability
- emotional consequence
- audio/display requirements
- Engine capability requirements
