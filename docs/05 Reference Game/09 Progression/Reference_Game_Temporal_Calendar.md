## Time Scales

| Scale | Use | Example |
|---|---|---|
| minutes | ambient slime behavior and short routines | slime naps, blinks, looks away |
| hours | NPC movement and environmental change | shop closes, town empties |
| day/night | form pressure and world tone | nocturnal behavior, vampire tendency |
| specific clock time | rituals and rare events | midnight gathering |
| multi-day | relationship memory and slow mystery | routine familiarity, recurring absence |

## Design Rules

- Rare events should leave traces if missed.
- Midnight events should be meaningful, not daily chores.
- Time-gated content should reward curiosity without punishing normal life.
- NPC schedules should produce clues before they produce requirements.
