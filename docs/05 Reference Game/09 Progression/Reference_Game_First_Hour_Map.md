# Reference Game First Hour Map

This note plans the first hour as a sequence of player understanding, not just events.

## Goals

By the end of the first hour, the player should understand:

- the slime already exists and is not a newborn pet
- the device remains emotionally alive when not actively played
- observation matters
- the world has routines
- the slime has agency
- hardware interactions are part of the fiction
- the game can shift from quiet observation to sudden intensity
