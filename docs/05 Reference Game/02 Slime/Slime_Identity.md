# Slime Identity

The slime is adaptive, intelligent, emotionally reactive, manipulative, socially performative, partially unknowable, and deeply expressive.

It is not a passive pet, slave creature, collectible, or stat block.

## Persistent Identity Markers

Across forms, the slime should remain recognizable through:

- eyes
- sloshy movement
- imperfect mimicry
- communication style
- emotional habits
- relationship memory

The player should always understand: "This is still my slime."

## Agency

The slime may:

- refuse things
- develop preferences
- react emotionally
- manipulate social situations
- remember routines
- learn tricks
- become suspicious, needy, proud, afraid, or stubborn
- change behavior over time

## Design Rule

The slime is a participant in the relationship.

Systems should make it legible without making it fully predictable.
