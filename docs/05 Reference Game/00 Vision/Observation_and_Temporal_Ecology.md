# Observation And Temporal Ecology

Observation is a core verb of the Reference Game.

The player learns by watching.

## Observation Behaviors

The game should reward:

- checking on the slime without immediate objective pressure
- watching NPC paths
- noticing who is missing
- following repeated behavior
- comparing day and night behavior
- recognizing ritual buildup
- revisiting places at strange times
- noticing environmental changes caused by light, motion, weather-like state, or player habits

## Temporal Ecology

Temporal ecology means the world uses real-world time, schedules, recurring behavior, environmental cycles, and ritual windows as part of play.

This includes:

- day and night cycles
- NPC routines
- recurring gatherings
- hidden social behavior
- midnight ritual events
- form tendencies shaped by routines
- pet mood drift
- slow mystery progression

## No Quest-Marker Dependency

The game should imply that something is happening without reducing mystery to objective markers.

The player should think:

"What are they doing out here?"

"Why is that place active now?"

"Why does my slime react differently tonight?"
