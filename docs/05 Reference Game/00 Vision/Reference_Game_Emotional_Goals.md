# Reference Game Emotional Goals

The Reference Game is designed around memory, atmosphere, emotional investment, ritual participation, companionship, curiosity, and long-term familiarity.

## Desired Player Memory

The ideal memory is:

"I stayed awake at midnight with my slime to witness something strange."

Not:

"I completed the content."

## Emotional Arc

The long arc should include:

- initial familiarity with virtual pet language
- surprise when that language mutates
- slow attachment to the slime as an agent
- routine and shared habits
- curiosity about the world
- fear of losing the slime
- rare rupture or death
- desperate ritual participation
- uncertain reunion
- rediscovery of continuity

## Emotional Continuity

The player should not only protect progress.

The player should care whether the slime remains recognizably itself.
