# Reference Game Structural Overview

The PeepShow Reference Game is the canonical demonstration title for the PeepShow platform.

It proves that ultra-low-power embedded hardware, a persistent display, physical sensors, ambient simulation, tactile interaction, emotional creature design, realtime microgames, and slow observational worldbuilding can combine into a new category of interactive experience.

The intended fantasy is not that the player launches a game app.

The intended fantasy is that a living fictional organism inhabits the physical handheld artifact.

## Structural Components

The Reference Game combines:

- persistent pet simulation
- turn-based RPG and roguelike structure
- Zelda/Metroid-style exploration progression
- realtime combat and ritual bursts
- environmental adaptation
- observational mystery systems
- temporal social simulation
- occult ritual systems
- form-based traversal and social access
- hardware-driven interaction mechanics
