## Questions To Revisit

- Which ideas create recurring player memories rather than one-off spectacle?
- Which ideas make the slime feel agentic?
- Which ideas create a reason to observe the world at unusual times?
- Which ideas demonstrate hardware without feeling like hardware demos?

## Promotion Checklist

Promote an idea when it has at least two of:

- a clear emotional purpose
- a runtime mode
- a related slime behavior
- a place, NPC, ritual, form, or enemy
- a hardware/Engine capability need
- a progression role
