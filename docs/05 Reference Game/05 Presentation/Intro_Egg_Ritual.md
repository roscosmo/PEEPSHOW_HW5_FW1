# Intro Egg Ritual

The intro intentionally imitates classic low-resolution virtual pets and egg-hatching rituals.

The player should initially think:

"I know what this is."

## Fake Retro Presentation

The opening may use:

- chunky LCD-era graphics
- familiar egg framing
- simple ceremony pacing
- nostalgic virtual pet language
- minimal dialogue

## Subversion

The egg appears to hatch.

Instead:

- a hand appears
- cracks the egg into a frying pan
- cooks it
- feeds it to the slime

This establishes:

- the slime already existed
- the protagonist already knows the slime
- this is not a birth narrative
- the game understands virtual pet rituals and mutates them

## Immediate Transition

The intro should be short.

After the subversion, the game should quickly move into:

- basic player/slime context
- a small home or world scene
- immediate ambient pet existence
- early [[Static_Exploration_Runtime]] access

When the player stops interacting, the game should naturally fall into [[Ambient_Pet_Runtime]].
